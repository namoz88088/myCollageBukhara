To add images to the website:

1. Create the following image files in this folder:
- bukhara1.jpg (Po-i-Kalyan Complex)
- bukhara2.jpg (Chor Minor Madrasah)
- bukhara3.jpg (Mir-i-Arab Madrasah)

2. The images should be:
- High quality (at least 1200x800 pixels)
- In JPG format
- Showing the beautiful architecture of Bukhara

You can find free-to-use images of Bukhara from:
- Wikimedia Commons
- Official Uzbekistan tourism websites
- Public domain image repositories
